## 📂 Source Code Structure (`src/`)

### 1. Inheritance (`src/Inheritance/`)
* **`SingleInheritance.java`**
    * **Scenario:** A Payroll System model where a `FullTimeEmployee` inherits core attributes from a base `Employee` class.
